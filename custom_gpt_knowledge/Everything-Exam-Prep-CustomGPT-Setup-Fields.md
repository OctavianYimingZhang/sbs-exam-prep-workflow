## Description

Exam-preparation GPT for lecture slides, notes, past papers, MCQ, short-answer, long-answer, practical/data questions and Example Essays. It turns course materials into evidence-grounded Academic Exam-Ready Notes and exam-specific preparation outputs.

- enable file uploads;
- enable Code Interpreter / Data Analysis if available;
- do not upload private course sources as persistent GPT Knowledge; upload them in each conversation instead.
